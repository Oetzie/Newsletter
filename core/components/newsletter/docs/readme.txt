## Requirements
* MODX version 2.5.0 or newer has to be installed.
* The user needs to have the "newsletter" permissions.
* The user administrator to have the "newsletters_settings" permissions.
* Form component (https://github.com/Oetzie/Form) required for the (un)subscribe actions.

We greatly value your feedback, feature requests and bug reports. Please issue them on GitHub (https://github.com/Oetzie/Newsletter/issues/new).
